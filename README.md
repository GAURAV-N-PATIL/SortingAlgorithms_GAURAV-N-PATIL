# SortingAlgorithms_GAURAV-N-PATIL
## Project Title
Selection Sort and Insertion Sort Implementation in C

---

## Overview
This project contains the implementation of two fundamental sorting algorithms:
- Selection Sort
- Insertion Sort

Both algorithms are implemented in the C programming language and demonstrate how elements are sorted step by step. The program displays the array after each pass or insertion, making the working of the algorithms easy to understand.

---

## Algorithms Implemented

### 1. Selection Sort
Selection Sort works by repeatedly selecting the minimum element from the unsorted portion of the array and placing it in its correct position.  
After each pass, the current state of the array is displayed.

### 2. Insertion Sort
Insertion Sort works by taking one element at a time and inserting it into its correct position within the already sorted portion of the array.  
The array is displayed after each insertion step.

## Features
- Takes array input from the user
